One inbox, every machine

Run the bridge on a desktop, a laptop, a mac mini in a closet — the client folds them into one list, sorted by what needs you.

Heads up. This web client needs your bridge reachable over HTTPS. The easiest path is tailscale serve --bg --https=443 http://localhost:<port> on the machine running rove-bridge. The app walks you through the full setup when you launch it.
